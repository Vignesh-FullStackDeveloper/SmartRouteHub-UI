# Widgets Module

This module contains reusable UI components used throughout the application.

## Widgets

- **OrgHeader**: Displays organization logo, name, and user role
- **PrimaryButton**: Primary action button with loading state
- **SecondaryButton**: Secondary action button with loading state
- **StatCard**: Card for displaying statistics with icon and value
- **InfoCard**: Card for displaying information with optional actions

## Usage

```dart
import 'package:school_bus_tracker/widgets/org_header.dart';
import 'package:school_bus_tracker/widgets/primary_button.dart';
```

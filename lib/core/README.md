# Core Module

This module contains core utilities, constants, theme configuration, and shared functionality used throughout the application.

## Structure

- **constants/**: Application-wide constants (API endpoints, default values, etc.)
- **theme/**: Material 3 theme configuration for light and dark modes
- **utils/**: Utility functions (validators, helpers, etc.)

## Usage

Import core utilities as needed:

```dart
import 'package:school_bus_tracker/core/constants/app_constants.dart';
import 'package:school_bus_tracker/core/theme/app_theme.dart';
import 'package:school_bus_tracker/core/utils/validators.dart';
```
